تحديث X-Minute Phase 2

هذا التحديث مبني على آخر نسخة شغالة بعد إصلاح AWS ونجاح X-Minute Phase 1.

ما تم إضافته:
- تحليل X-Minute داخل Cloudflare Worker بعد قراءة الملف.
- حساب Date range وعدد الصفوف وعدد التوربينات والمتغيرات.
- تصنيف المتغيرات Categories.
- ملخص لكل Turbine.
- إحصائيات للمتغيرات الرقمية.
- Temperature hotspots.
- Exceedances حسب الحدود المستخدمة في الواجهة.
- عرض قسم مؤقت داخل X-Minute باسم X-Minute Backend Analysis للتأكد أن الحسابات راجعة من الباك إند.

ملاحظة: القسم الجديد مؤقت للتأكيد مثل مرحلة AWS، ويمكن دمجه أو إخفاؤه لاحقًا بعد الاختبار.
